\doxysubsection{Detailed Description}
A graphical pixel widget that inherits from gl\+Widget and \doxylink{classgl_video_memory_plot}{gl\+Video\+Memory\+Plot}. 

This class represents a single pixel in a graphical user interface. It provides multiple constructors for different ways of initializing the pixel\textquotesingle{}s position and color. It also overrides the Redraw method to plot the pixel on the screen.
